# Setup & Installation
 - In clientSSR, 
 - npm install 
 - npm run dev on localhost:3000
 - In server,
 - add your own database URI in .env (tables will be created automatically if the first api request is made)
 - pip3 install -r requirement.txt
 - run python3 app.py on localhost:8080
